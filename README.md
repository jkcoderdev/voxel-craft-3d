# Voxel Craft 3D

A voxel sandbox game.

![Voxel Craft 3D Screenshot](achievements/5-block-placing-and-removing.png 'Voxel Craft 3D Screenshot')

## Getting Started

Install the dependencies:

```shell
npm install
```

Start the development server:

```shell
npm run dev
```

Build the project:

```shell
npm run build
```
